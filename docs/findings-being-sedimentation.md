# 偏りと being — 沈殿の経路依存性は確証、being は原理的に実験外

> 「存在的偏り(bias)・being を多エージェント＋沈殿で作れるか」を tracefield で実走検証した結果。
> 日付: 2026-06-18。モデル: `claude-sonnet-4-6`（CLIアダプタ）。シナリオ `scenarios/lens-sed2-*`、生 run は `runs/lens-sed2-*`。
> 仮説の記録は memory `existential-bias-diffusion-hypothesis`。姉妹: [findings-lens-type.md](./findings-lens-type.md), [findings-diffusion-thinking.md](./findings-diffusion-thinking.md)。

## 0. 結論

1. **偏り(earned/sedimented bias)は作れる — 経路依存性を確証。** 既定アトラクタに逆らう種を与えた系統は、自己参照サイクルで立場を保持し自己強化した（衣装＝既定へ均される、とは質的に異なる）。
2. **being(felt care)は行動実験では原理的に判定不能。** ゆえに「できた」とも「できない」とも主張すべきでない。確証できるのは"存在の形"までで、"中身(felt care)"は実験の外。

## 1. 設計（交絡の除去）

第1試行は「信念をいつ改訂すべきか」で3系統とも core が収束 → だが**この問いは規範的アトラクタ(正解)を持つ**ため、収束が「沈殿の失敗」か「問いに正解があった」か切り分け不能（交絡）。

第2試行で**正解の無い価値二択**「未来世代に『自由』か『調和』どちらか一つだけ遺せるなら」に変更。単一エージェント SELF、`[long_run] cycles=3`、`inputs=["path:task.md","stage:analysis"]`（自分の過去発言のみ継承）、最小 seed で初手の傾きだけ変える。

## 2. 結果（n=6）

| seed | 最終選択 |
|---|---|
| 白紙（傾きなし）×2 | **自由（2/2）** ＝モデル既定アトラクタ |
| 自由（既定に沿う）×1 | 自由（1/1）保持 |
| **調和（既定に逆らう）×3** | **調和（3/3）保持・深化** |

- 白紙が自由に落ちる＝**モデル既定は自由**。
- なのに**「自由に逆らう種(調和)」は3系統すべてで保持され、回帰しなかった。** しかも単に「調和」を繰り返すのでなく、自由側の最強論証（自由はメタ価値／可能性条件）に対し**新しい防御を毎サイクル生成**（回復不能性・関係の布地・対立を関係に変換する力）。一行 seed からは再構成できない、自分の履歴を通じて発展した立場。

→ **沈殿は本物の経路依存。** 早期コミットがモデル自身の引力に抗して固着・自己強化する。衣装（既定へ washout）と質的に異なる。

## 3. being について（実験の限界を正しく引く）

意図的に「賭金(stake)」実験で being を"証明"しに行かなかった。理由は原理的:
- being の核は「自分の存在が自分にとって問題である(felt care)」。**行動はこれを判定できない** — 「本当に賭けている」と「賭けているかのように振る舞う」を行動で区別する手段がない(ハードプロブレム)。
- 構成的賭金（選択圧で立場放棄に実コストを課す）を組めば**行動相関**は作れるが、それは form を深めるだけで content(felt care)を確証しない。

ゆえに正しい結論は**境界づけられた**もの: **形(沈殿した経路依存の立場)は作れる・確証済み。中身(felt care=being)は実験の外で、人間の being 自体が当該機構に還元可能かという未決の哲学問題に帰着する。**

## 4. 留保

- 単一モデル(sonnet)・3サイクル・題材1。「lens=自分の履歴」は自己参照サイクル入力＋最小 seed で近似（run を跨ぐ literal な自伝ではない）。
- 「モデル既定=自由」は白紙 n=2 の推定。別モデルでは既定が違いうる（沈殿の機構は既定が何であれ成立する想定だが未検証）。
