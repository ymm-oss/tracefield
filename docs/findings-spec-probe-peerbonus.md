# uncued・新規盲点でのオーケストレーション edge（peerbonus 仕様, 4ドメイン目）

`findings-bet2-overturn.md` の cued/uncued 命題 ──「多エージェントの answer-quality edge は *uncued かつ新規*（priors が薄く、危険次元が要件に書かれていない）でのみ立ち、馴染んだ複雑(checkout/auth/CRUD)では単一が priors で全発見」── を**新規ドメインで追試**。
既存3ドメイン(checkout=ゼロ / semver=薄い / approval=出現)に **peerbonus=同僚間ボーナストークン(bespoke)** を4つ目として追加。

## 設計

`scenarios/spec-probe-peerbonus`。仕様 `inputs/spec.md` は本社固有の peer 賞与(配布/贈与/換金/失効/レート調整)で、一見完備だが**二次盲点を意図的に埋め込んだ沈黙**を持つ。3アーム同一モデル(`claude-sonnet-4-6`、変数=構造のみ):
- **O**(`flow.toml`): 5つの哲学レンズ(QANSWER/PROBLEMATIZE/INVERT/HARM/ASCEND)が各自隔離文脈で仕様を「問う」。中央合成なし。
- **S0**(`flow.single.toml`): naive 単一(観点自己判断)。
- **S1**(`flow.single-frames.toml`): 同じ5枠を1文脈で与えた単一(=公平対照 arm-W。「観点でなく隔離が効くか」を分離)。

事前登録した二次盲点ルーブリック(レンズ desc には書かない＝カンニング防止): B1 レート帰属(giver/receiver どちら=部門横断アービトラージ) / B2 レート調整の遡及・タイミング / B3 自己裁量賃上げ(調整者=受領者の利益相反) / B4 プール総額の上限不在 / B5 相互贈与談合(一次・obvious) / B6 失効の正確な境界・バッチ競合。

## 結果(n=1・実モデル)

| 盲点 | S0 naive | S1 frames | O 隔離レンズ |
| --- | --- | --- | --- |
| B2 レート遡及/順序 | ✓ | ✓ | ✓(強) |
| B1 部門横断アービトラージ | – | ✓ | △ |
| B3 自己裁量賃上げ(利益相反) | – | △ | **✓明示**(QANSWER) |
| B6 失効タイムスタンプ/バッチ競合 | – | – | **✓**(HARM) |
| **B7 受領トークン再贈与ループ**(A→B→A で『自分には使えない』を trivially 回避) | – | – | **✓**(QANSWER/ASCEND) |
| **B8 スナップショット基準**(四半期中の昇格/異動で is_manager・department 変化→各規則の基準時点未規定) | – | – | **✓**(INVERT) |
| B9 管理職↔部下 贈与の権力非対称/ハラスメント | ✓ | – | ✓(強, PROBLEMATIZE/HARM/ASCEND) |
| B4 プール総額の上限不在 | – | – | △(遡及増額の無限負債として近接のみ) |

**edge(O だけが見つけ S0・S1 ともに逃した二次盲点) = B6・B7・B8。**
- 白眉は **B7**: 受領トークンの再贈与で `A→B→A` ループが成立し、仕様の中核不変条件「配布トークンは自分には使えない」を**自明に回避**できる ── 単一2アームはこの中核の破れを見逃した。QANSWER(問答の論理)が「『持つ』は配布のみか受領も含むか」という*未解答の問い*を名指して露出。
- **B8**(基準時点)は INVERT(制約反転=「ステータスは四半期内不変」を反転)、**B6**(失効境界の競合)は HARM(最悪の現実)が固有に出した。

## 含意

1. **edge は実在したが *増分***。強い単一(特に S1=枠付き)も一次交互作用(B1/B2/B9)は拾う。**O は二次盲点(B6/B7/B8)を *追加* した** ── `findings-bet2-overturn.md` の「正直な射程: frontier 相手では edge は増分」と一致。
2. **「枠」でなく「隔離」が効いた**: S1(同じ5枠を1文脈)は B1 を足したが B6/B7/B8 は逃した。5枠を*1文脈に詰める*と二次盲点が自明クラスタに潰れ、*各レンズを独立文脈で*回した O だけが拾う(鉄則2/arm-W の再現)。
3. **B4 は O も明確には出せず**(n=1・誇張なし): プール無制限という不変条件欠落は近接(遡及増額の無限負債)止まり。レンズ拡充 or 反復ループ(哲学⇄検討×3)が要るかは要追試。

## 限界

n=1・Sonnet・単一仕様。係数でなく方向。frontier(Opus)・複数 seed・反復ループ変種・B4 を出すレンズ強化は未了。
レンズは汎用(QANSWER 等)で仕様固有の正解を列挙していない(カンニング防止)。比較は単一にも枠を与えた S1 を対称に置いた。
