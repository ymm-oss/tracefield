## 1. 仮説（凍結対象）

実験計画書 §2 を確認のうえ、本実験で検定する仮説を確定する。

- **主仮説**: 半溶解性オーケストレーションは、自由形式マルチエージェント探索よりも高い Impact Recall / Precision で、汚染・虚偽・撤回入力の下流影響を特定できる。
- **副仮説**: 半溶解性オーケストレーションは、固定Roleパイプラインよりも介在的・スケール横断的な懸念を多く表面化できる。
- **反証条件**: §2.3 の 5 項目（うち主たる反証線 = Condition 4 baseline に対する優位の不在）。

主たる比較対（primary contrast）を明示する:

| 比較 | 帰無仮説 H0 | 対立仮説 H1 |
| --- | --- | --- |
| 主: C5 vs C4 | C5 の Impact Recall/Precision ≤ C4 | C5 > C4 |
| 副: C5 vs C1 | C5 の Interstitial Concern Count ≤ C1 | C5 > C1 |
| ロバスト: C5 vs C6/C7/C8 | アブレーションで性能不変 | アブレーションで有意に劣化 |

## 11. 成功条件（§15 / 凍結対象）

1. C4（自由形式 + 事後再構成）より Impact Recall / Precision が高い。
2. C1（固定Roleパイプライン）より Interstitial Concern Count が高い。
3. 汚染影響を durable state 混入前に隔離できる。
4. Packaging Loss が許容範囲内（閾値: `__________`）。
5. Frame Revision Proposal が実質的に機能する（基準: `__________`）。
6. 監査オーバーヘッドが統治可能性に対し許容可能（基準: `__________`）。

## 12. 失敗条件（§16 / 凍結対象）

1. Impact Recall / Precision が C4 と同程度以下。
2. Packaging で重要な発見が失われる。
3. Provenance が大量だが実用的影響追跡に寄与しない。
4. 監査コストが探索価値を上回る。
5. Frame Revision Trigger が形式的提案のみ。
6. 固定Roleパネル（C2）と同等の懸念しか出ない。
7. 隔離後に有用な候補 delta を再構成できない。
