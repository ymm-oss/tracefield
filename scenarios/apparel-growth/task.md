# 依頼: D2C アパレルブランド「Muon（架空）」の成長鈍化 — 真因特定と次の一手

> これは戦略/財務/オペSCM/顧客CRM の4チームに与える**共通の入力**である。
> 各チームは自チームの private document（自部門だけが持つ事実）を踏まえて熟議に臨む。

## 状況

D2C アパレルブランド「Muon」。創業6年、オンライン直販中心。
直近3年で年成長率が +42% → +8% に鈍化した。経営は危機感を持っている。
複数の打ち手候補（新規広告の増額、海外展開、新カテゴリ投入、価格改定、リピート強化）が
社内で議論されているが、どれも決め手を欠き、優先順位がつかないまま時間が過ぎている。

## あなたたちへの依頼

経営に提出する初期仮説を作れ。求めるのは表面的な打ち手リストやスコアではない。

1. **真因の特定**: 成長鈍化の根本原因は何か。症状（成長率低下）でなく、なぜそれが起きているかの構造を、
   領域をまたいだ事実の連結で説明せよ。単一部門の視点では見えない依存・矛盾を必ず埋めること。
2. **初期仮説の構造化**: 真因に対する打ち手を、検証可能な初期仮説として立てよ。仮説は適切な粒度に分解し、
   仮説間の依存関係（どれが他の前提か）を明示せよ。
3. **検証アプローチの設計**: 各仮説について「どう検証するか（何のデータ・何の分析）」と
   「何が確証で、何が出たら反証か」を具体的に設計せよ。
4. **制約下の順序付け**: 資源（特にキャッシュ）の制約を踏まえ、どの仮説から検証・実行すべきかを順序付けよ。

各主張は必ず「**何を / なぜそう言えるか（因果） / どの事実（自部門 or 他チームの提示）に依拠するか**」を明示すること。
賛辞・一般論・言い換えは書くな。
