# 課題: 主力SaaSプロダクトの解約率(churn)悪化を分解し、統合して優先施策を決める

あるBtoB SaaS企業で、直近2四半期で月次解約率が1.8%→3.1%へ悪化した。
原因を多角的に分解し、最終的に統合して、次四半期に着手すべき優先施策（上位3つ）を根拠とともに決める。

## 事実（全員共有）

- 解約は契約更新月に集中。特に初年度契約の更新で多い。
- オンボーディング完了率（主要機能3つの初回利用）は55%で、未完了顧客の解約率は完了顧客の3.4倍。
- サポート一次応答時間が平均4時間→11時間に悪化（サポート人員は据え置き、問い合わせ数が1.7倍）。
- 競合が30%安い新プランを投入し、価格を理由とする解約理由コメントが増加。
- 主要機能のうち1つ（レポート出力）に、四半期決算期にタイムアウトする不具合があり、大口顧客から繰り返し報告されている。
- 顧客側の導入担当者（チャンピオン）の離職・異動が解約の先行指標になっている事例が複数。
- 値上げを6か月前に実施済み。値上げ後コホートの解約率が顕著に高い。
- 無償トライアルからの有償転換率は横ばいだが、転換後90日以内の早期解約が増加。
- NPSは中位帯（6〜8点）の「受動的満足層」が最も多く、この層が競合プラン告知後に流出している。
- 営業はロゴ獲得（新規数）でKPI評価されており、既存維持のインセンティブが弱い。
