# 関数仕様: `merge_intervals(intervals)`

整数の閉区間 `[start, end]` のリストを受け取り、**重なるか隣接する**区間をマージして返す。

## シグネチャ
```python
def merge_intervals(intervals: list[list[int]]) -> list[list[int]]:
```

## 契約
- 各区間は `[start, end]`（`start <= end` の整数, **閉区間** = 両端を含む）。入力の各区間は well-formed（`start <= end`）とみなしてよい。
- **重なる区間、および隣接する（接する: 前の `end` == 次の `start`）区間はマージする。**
  - 例: `[1,3]` と `[3,5]` → `[1,5]`（端点を共有＝接するのでマージ）。
  - 例: `[1,3]` と `[4,5]` → そのまま2つ（間が空いている）。
- **出力**は `start` の昇順にソートされ、互いに重ならず接しもしない（**disjoint**）区間のリスト。
- **入力は順不同でよい**（ソートは関数の責任）。
- **空入力 → 空出力。**

## 補足
- 入力リスト自体や各区間を破壊的に変更しないことが望ましい。
- 同じ区間が複数回現れてもよい。
