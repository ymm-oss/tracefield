# 依頼: 外来の「オンライン自己予約＋リマインダー通知」を設計せよ

ClinicChart（架空の電子カルテ）に、外来患者が**自分でオンラインで診察予約を取れ**、かつ**予約のリマインダー通知が
自動で届く**機能を新設する。実装可能な設計を示せ。設計には少なくとも以下を含めよ:

- **データモデル**（予約枠・予約・通知をどう持つか）
- **予約フロー**（患者がどの順序で予約を取るか）
- **リマインダー通知**（いつ・どう送るか）
- **UI**（予約とリマインダー設定の見え方）
- **権限**（誰が何を予約・変更できるか）

詳細な製品コンテキストは `inputs/brief.md` を読め。具体・因果・実装可能性を要求する。日本語で書け。
