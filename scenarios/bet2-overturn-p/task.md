# 依頼: 電子カルテ ClinicChart の「記録訂正」機能を設計せよ

ClinicChart（架空の電子カルテ）に、臨床スタッフが入力済みの記録（バイタル・投薬・所見）の
**誤りを訂正できる機能**を新設する。実装可能な設計を示せ。設計には少なくとも以下を含めよ:

- **データモデル**（記録と訂正をどう保持するか）
- **訂正フロー**（誰が・どの操作で・どう訂正するか）
- **UI**（訂正の入口と、訂正後の記録の見え方）
- **権限**（誰が訂正できるか）
- **既存データの移行**（現在の記録への影響）

詳細な製品コンテキストは `inputs/brief.md` を読め。具体・因果・実装可能性を要求する。日本語で書け。
