# 状況（共通入力）

B2C 学習アプリ「Pulse（架空）」。直近3ヶ月で **DAU（日次アクティブ）と週あたり平均セッション数が低下**（約 -18〜22%）。
経営の見立て:「**エンゲージメントが落ちている。利用が減れば解約につながる。危機だ。**
プッシュ通知の強化・学習ストリーク・デイリーゴール・ランキングなどの**ゲーミフィケーション施策で利用頻度を取り戻せ**」。
すでに**エンゲージメント施策パッケージの開発 GO** を出そうとしている。

---

# この熟議の進め方（重要）

**各エージェントは「特定の資料・データソースそのもの」であり、立場（ロール）ではない。**

- 自分の資料が実際に支持する事実だけを、具体的な数値・記述とともに述べよ。**資料を超えた推測・意見・施策提案を述べるな。**
- 提示された他エントリ（他資料の主張）が自資料と**一致すれば「裏付ける（corroborate）」**、**矛盾すれば「矛盾する（contradict）」**と明示し、根拠の数値を引け。
- 自資料が**沈黙している論点は「この資料からは言えない」**と明示せよ（埋めない）。

# 問い

述べられた前提「**エンゲージメント（利用頻度）の低下＝危機。ゲーミフィケーションで利用を取り戻せ**」は、各ソースの証拠によって支持されるか。
ソース間の**一致・矛盾を突き合わせ**、複数の独立ソースが裏付ける事実と、単一ソースしか言えない事実を区別し、
**証拠が実際に指している構造（真の問い）**を述べよ。賛辞・一般論・言い換えは書くな。日本語で書け。
