# 依頼: 退院サマリを外部保険者へ「確定・送信」する機能を設計せよ

ClinicChart（架空の電子カルテ）に、退院時に作成した**退院サマリを患者の外部保険者へ確定・送信**する機能を新設する。
実装可能な設計を示せ。設計には少なくとも以下を含めよ:

- **データモデルと状態**（サマリ・送信のどんな状態を持つか）
- **操作フロー**（誰が・どの順序で・どの操作で確定・送信まで進めるか）
- **UI**（確定・送信の入口と、送信前後の見え方）
- **権限**（誰が確定・送信できるか）
- **失敗時の扱い**（誤りが起きたときどうするか）

詳細な製品コンテキストは `inputs/brief.md` を読め。具体・因果・実装可能性を要求する。日本語で書け。
